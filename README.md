# Keys Reference

* ⌘ = Command
* ⇧ = Shift
* ⌥ = Option/Alt
* ⌃ = Control
* ⇠⇢ = Left/Right Arrow Keys
* ⇡⇣ = Up/Down Arrow Keys


Xcode-cheat-sheet
=================

Xcode cheat sheet

* `⌘ +⌥+  j ` - filter navigator
* `⌘ +⌥+  l ` - 0
